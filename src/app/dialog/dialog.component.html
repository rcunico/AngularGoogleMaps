GGC's Data on Display

Description

Our team is working on an interactive mapping application as part of our Software Development course with Dr. Anca Mihu. The application will be accessed as a website through Georgia Gwinnett College (GGC) that will be available to everyone. The application displays a graphical representation of raw data, provided by GGC departments and will be confined to the Northeast Metro Atlanta region. A user can navigate within the region, which will dynamically update the data points displayed in the interactive map. Additionally, a user will be able to customize the data types displayed through selection filters corresponding to different categories. Another feature allows the user to select an individual data point, which will display a popup window with details for the point. The map will provide a convenient visualization tool for a wide variety of historical and demographic data in the Gwinnett County region.
